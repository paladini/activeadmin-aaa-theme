## Recent Decisions (Last 60 days)

### AD-001: Tech Stack Choice (2026-07-14)

**Decision:** Create a lightweight Rails Engine gem delivering vanilla SCSS/CSS variables.
**Reason:** Active Admin views are generated via Arbre DSL on the server-side. Overriding raw HTML is brittle across versions. Providing class-level and custom-property overrides via CSS is the most lightweight, robust, and future-proof way to customize the UI.
**Trade-off:** We are constrained to the HTML structure generated by Active Admin, but we can completely restyle it via CSS Grid, Flexbox, and modern CSS features like `:has()`, relative color syntax, etc.
**Impact:** Easier installation, 100% compatibility with Sprockets/Importmaps, and simple maintenance.
